Rich Text to PptxGenJS Converter

Convert Markdown and HTML rich text to PptxGenJS-compatible format

Input (Rich Text)

Supports: **bold**, *italic*, <u>underline</u>, ~~strikethrough~~

Output (PptxGenJS Format)

Ready to use with PptxGenJS addText() method

Usage Instructions

Supported Formats

  • Bold: **text** or __text__
  • Italic: *text* or _text_
  • Underline: <u>text</u>
  • Strikethrough: ~~text~~
  • Line breaks: Empty lines or newlines

Usage with PptxGenJS

const richTextArray = /* paste output here */;
slide.addText(richTextArray, {
    x: 1, y: 1, w: 8, h: 4
});
